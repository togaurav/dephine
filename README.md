# Dephine

Google dictionary in terminal

## Installation

	$ gem install dephine

## Usage

	$ dephine [word]

## What it looks like

	$ dephine anything
	anything

	pronoun /ˈenēˌTHiNG/
	  Used for emphasis
	    - I was ready for anything

	  Used to indicate a range
	    - he trains anything from seven to eight hours a day
